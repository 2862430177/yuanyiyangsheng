## 平台简介

基础架构基于若依，版本[RuoYi-Vue3](https://github.com/yangzongzhuan/RuoYi-Vue3)

## 开源协议：
GPL v3

## 业务功能：

已实现功能
1. 会员管理，包括增删改查等功能
2. 会员卡管理，包括会员卡充值功能，会员卡提成配置，介绍员工绑定功能
3. 员工管理
4. 店铺设备管理
5. 店铺提供的服务项目，支持按固定金额和固定比例提成配置
6. 店铺提供的服务产品，支持按固定金额和固定比例提成配置
7. 消费结算，指定服务员工和会员结算，支持现金结算和会员卡结算
8. 店铺维度、服务项目维度、服务产品维度，数据统计
9. 员工维度，员工提成统计
10. 营销管理：包括短信模版维护，定时短信发送任务创建，短信发送记录功能

## 计划功能：
1. 短信模版创建修改删除，调用腾讯云短信接口
2. 发送短信调用腾讯云短信接口
